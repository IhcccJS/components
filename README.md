## 发布

版本语义

- 大量更新，可能包含不兼容情况，更新第一位
- 新组件或功能，更新第二位
- 优化或问题修复，更新第三位

其它后缀

- beta 公开测试：`0.1.0-beta.1`
- alpha 内网测试：`0.1.0-alpha.1`
- local 本地测试：`0.1.0-local.1`


```bash
npm publish
```
